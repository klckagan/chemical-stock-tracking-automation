# Chemical Stock Tracking Automation

A low-code automation solution developed during my Information and Operational Technologies internship at Tüpraş.

The project was designed to monitor chemical stock levels stored in SharePoint and send automated email notifications when inventory values fell below predefined threshold levels.

## Features

- SharePoint-based stock tracking
- Scheduled stock level checks
- Threshold-based monitoring
- Automated email notifications
- Reduced need for manual inventory control
- Low-code workflow automation

## Technologies

- Microsoft Power Automate
- SharePoint
- Microsoft 365
- Email Automation
- Robotic Process Automation (RPA)

## Workflow

SharePoint List
      ↓
Scheduled Power Automate Flow
      ↓
Check Stock Threshold
      ↓
Send Email Alert
